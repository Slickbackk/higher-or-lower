# Higher or Lower - Number Guessing Game

A simple Python number guessing game where the computer picks a random number between 1 and 100, and you try to guess it with hints.

## How to Play

```bash
python src/guessing_game.py
```

- The computer picks a secret number (1-100)
- You enter guesses and get "Too high!" or "Too low!" hints
- Try to find the number in the fewest attempts
